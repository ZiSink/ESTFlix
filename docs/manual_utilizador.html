1. Introdução

1.1 O que é a ESTFlix?

A ESTFlix é uma plataforma web de streaming de conteúdos multimédia desenvolvida no âmbito da Unidade Curricular de Programação Web do curso CTPSI do IPS. Inspirada em plataformas como Netflix, permite aos utilizadores explorar um catálogo de filmes e séries, guardar favoritos, acompanhar o histórico de visualizações e receber recomendações personalizadas.

A aplicação é totalmente web, não requer instalação no computador do utilizador final, e é acedida através de qualquer browser moderno via http://localhost:3000.

1.2 Para quem é?

  • Utilizador comum: explora o catálogo, escolhe um perfil, guarda favoritos e vê o histórico.
  • Administrador: gere o catálogo (criar, editar, apagar conteúdos e categorias) através do painel de administração.

1.3 Funcionalidades disponíveis

  • Registo e login com e-mail e password
  • Gestão de múltiplos perfis por conta
  • Navegação e pesquisa no catálogo (por tipo, categoria e texto)
  • Adicionar/remover conteúdos da lista de favoritos
  • Registo automático de histórico de visualizações
  • Recomendações personalizadas com base no histórico
  • Área de administração: CRUD completo de conteúdos e categorias

2. Instalação e Configuração

2.1 Requisitos

  • Node.js versão 18 ou superior — node --version
  • MySQL 8.x (via MAMP, XAMPP, ou instalação direta)
  • Browser moderno (Chrome, Firefox, Edge, Safari)

2.2 Passos de instalação

1

Iniciar o MySQL

Abrir o MAMP e clicar em Start Servers. Aguardar até o ícone do MySQL ficar verde.

2

Criar a base de dados

Na pasta do projeto, executar o script de criação das tabelas:

mysql -u root -proot < schema.sql
3

Popular a base de dados

Executar o script de seed para criar utilizadores demo e o catálogo inicial:

node seed.js

Serão criados dois utilizadores de demonstração:

  • admin@estflix.test / admin — perfil ADMIN
  • viewer@estflix.test / viewer — perfil USER
4

Instalar dependências (primeira vez)

npm install

2.3 Iniciar o servidor

node server.js

O terminal deverá mostrar:

ESTFlix a correr em http://localhost:3000
Credenciais demo: admin@estflix.test / admin  |  viewer@estflix.test / viewer
Abrir o browser em http://localhost:3000. A aplicação não funciona corretamente se aberta diretamente como ficheiro (file://) — necessita sempre do servidor Node.js.

3. Guia de Utilização

3.1 Registo de conta

Ao aceder à aplicação pela primeira vez, é apresentada a página de login. Para criar uma nova conta:

1

Clicar em "Create account"

No formulário de login, clicar no separador ou botão Create account.

2

Preencher os dados

Introduzir nome, endereço de e-mail e password. Todos os campos são obrigatórios.

3

Confirmar

Clicar em Register. A conta é criada, o login é feito automaticamente e o utilizador é redireccionado para a página inicial.

localhost:3000/login.html
ESTFlix Stream what you love. Create account Name [ Maria Silva ] E-mail [ maria@example.com ] Password [ •••••••• ] [ Register ] Already have an account? Sign in

3.2 Login

1

Introduzir credenciais

Preencher o e-mail e a password no formulário de login.

2

Clicar em "Sign in"

Se as credenciais estiverem corretas, o utilizador é redireccionado para a página de seleção de perfil.

localhost:3000/login.html
ESTFlix Stream what you love. Sign in E-mail [ admin@estflix.test ] Password [ •••••• ] [ Sign in ]
Erro: Se as credenciais forem incorretas, é apresentada a mensagem "Invalid credentials." A password é sensível a maiúsculas/minúsculas.

3.3 Seleção de perfil

Cada conta pode ter múltiplos perfis (ex: "Admin", "Cinephile"). Após o login, é apresentada a página de seleção de perfil.

1

Escolher um perfil

Clicar em Use profile no cartão do perfil pretendido.

2

Criar perfil novo (opcional)

Preencher o nome no formulário Add profile e clicar em Create profile.

Criar categoria

  1. Preencher o campo Name no formulário de categorias.
  2. Clicar em Create category.
  3. A tabela atualiza-se automaticamente.

Editar categoria

  1. Clicar em Edit na linha da categoria. O formulário preenche-se com o nome atual.
  2. Alterar o nome e clicar em Save changes.
  3. Clicar em Cancel edit para cancelar sem guardar.

Apagar categoria

Clicar em Delete. Se a categoria tiver conteúdos associados, é mostrada uma mensagem de erro e a categoria não é apagada.

Erro: "Cannot delete category with associated contents." — remover ou reatribuir os conteúdos dessa categoria antes de apagar.

Criar / editar conteúdo

Funciona de forma análoga às categorias. Campos obrigatórios: título, descrição e URL da imagem. O elenco deve ser separado por vírgulas (ex: Christian Bale, Heath Ledger).

3.8 Logout

Clicar em Logout no canto superior direito. A sessão é destruída no servidor e o utilizador é redireccionado para a página de login.

Fechar o browser sem fazer logout não destrói a sessão imediatamente — o cookie dura 7 dias. Recomenda-se fazer logout explícito em computadores partilhados.

4. Situações de Erro e Comportamentos de Exceção

SituaçãoComportamento
Login com credenciais erradas Mensagem "Invalid credentials." no formulário. A página não recarrega.
Registo com e-mail já existente Mensagem "Email already in use.". Sugerir login com esse e-mail.
Aceder a /admin sem perfil ADMIN Mensagem de acesso negado com botão para trocar de perfil.
Aceder a /admin sem sessão Mensagem para fazer login primeiro.
Apagar categoria com conteúdos Erro 409 da API; mensagem visível no painel de admin.
Conteúdo não encontrado (URL inválida) Página de detalhe mostra "Content not found" com botão de retorno.
Clicar Play/Favorito sem perfil ativo Redireccionamento automático para login.html.
Servidor não está a correr O browser mostra erro de ligação. Verificar que node server.js está em execução.
MySQL não está a correr O servidor arranca mas todas as chamadas à API retornam erro 500. Iniciar o MAMP.
Pesquisa sem resultados Grelha vazia com mensagem "No titles match your search."
Lista de favoritos vazia Mensagem "Nothing saved yet." com botão para explorar o catálogo.
Sem histórico (recomendações) A secção de recomendações não aparece na home se não houver histórico.
